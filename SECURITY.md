# Security Policy

## Supported versions

This project is currently in active MVP development.

Security fixes will be applied to the `main` branch.

## Reporting a vulnerability

If you discover a security issue, please do not open a public issue first.

Instead, report it privately by contacting the repository owner at:

- `bentancourtfiorellanahir@gmail.com`

Please include:

- a clear description of the issue
- steps to reproduce it
- potential impact
- any suggested mitigation, if available

You can expect:

- acknowledgement as soon as possible
- a review of the report
- a best-effort update on remediation status

## Scope

At this stage, the most relevant areas include:

- API endpoints
- database configuration
- secrets handling
- dependency vulnerabilities
- ingestion pipelines and source handling

Thank you for helping keep the project safe.
